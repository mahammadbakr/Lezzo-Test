## Getting Started

1- clone source code:
```node
git clone https://github.com/mahammadbakr/Lezzo-Test.git
```

2 - Install:
```node
npm install
```

3- Run project:
  You can use one of these commands to run the project: 
  ```dart
  node index.js
  ```
 ```dart
  npm start
  ```
   ```dart
  nodemon index.js
  ```


## Add .ENV File

  create a file in project folder named ".env"
  and write this code inside it.
  ```dart
  PORT = 5000
  ```
